Formação Academica

Instituições responsaveis pela minha formação.

  • 2017-2019

    Colegio São Luis

    O Colégio São Luís é uma das instituições de ensino mais tradicionais e respeitadas de São Paulo, com uma rica história que remonta ao século XIX. Originalmente fundado em Itu, o colégio foi transferido para São Paulo em 1918, ocupando um edifício na Avenida Paulista, onde se tornou um símbolo da educação jesuíta na região. Administrado pela Companhia de Jesus, o colégio segue os princípios da pedagogia inaciana, que enfatiza a formação integral do aluno, desenvolvendo não apenas competências acadêmicas, mas também valores humanos e sociais!

  • 2021-2024

    Centro Universitário FEI

    A FEI (Faculdade de Engenharia Industrial), hoje conhecida como Centro Universitário FEI, é uma instituição de ensino superior que se destaca por sua tradição e excelência em engenharia e tecnologia. Fundada em 1945 pela Companhia de Jesus, a FEI tem um histórico de formação acadêmica voltada ao desenvolvimento industrial e social. Desde o início, ofereceu cursos inovadores como Engenharia Mecânica e mais tarde, a primeira Escola Superior de Administração de Negócios do Brasil, conhecida como ESAN.

Python

Python é uma linguagem de programação poderosa e de fácil leitura, amplamente utilizada em diversas áreas, incluindo ciência de dados, desenvolvimento web e machine learning. No meu Trabalho de Conclusão de Curso (TCC), que foca na criação de modelos 3D de construções civis a partir de imagens, utilizo Python para implementar algoritmos de visão computacional, aproveitando bibliotecas como OpenCV para capturar e processar dados visuais. Além disso, emprego técnicas de machine learning com Keras e TensorFlow, que me permitem desenvolver modelos de aprendizado profundo para análise e interpretação dos dados, melhorando a precisão dos resultados. Essa combinação de Python com frameworks de machine learning e algoritmos avançados possibilita uma abordagem inovadora e eficiente para a visualização e modelagem de estruturas, garantindo a excelência no meu projeto.

Java

Java é uma linguagem de programação versátil e robusta, amplamente utilizada no desenvolvimento de aplicações corporativas e web. Em meus projetos, aproveito as características de orientação a objetos (OOP) do Java para criar soluções modulares e reutilizáveis, facilitando a manutenção e a escalabilidade do código. Utilizando o framework Spring, consigo implementar práticas de desenvolvimento ágil e estruturação eficiente, aproveitando recursos como injeção de dependências e gerenciamento de transações. Esses projetos me permitem explorar a criação de APIs RESTful e a integração com bancos de dados, proporcionando uma base sólida para a construção de aplicações dinâmicas e de alta performance.

SQL

SQL é uma linguagem essencial para gerenciar e manipular dados em bancos de dados, e sua versatilidade se reflete no meu trabalho com diferentes tipos de sistemas de gerenciamento de bancos de dados, tanto relacionais quanto não relacionais. Já utilizei PostgreSQL para gerenciar dados estruturados, aproveitando suas robustas funcionalidades de consultas e transações. Além disso, tenho experiência com MongoDB, que me permite lidar eficientemente com dados não estruturados e semi-estruturados. Para orquestração de workflows de dados, apliquei o Apache Airflow, que facilita a automação e agendamento de tarefas de ETL (extração, transformação e carregamento). Também trabalhei com Apache Spark para processamento de grandes volumes de dados, integrando essas tecnologias em projetos que exigem análises complexas e em tempo real. Essa combinação de habilidades me proporciona uma sólida base para manipular e analisar dados de maneira eficaz, permitindo a construção de soluções escaláveis e eficientes.

Html

HTML (HyperText Markup Language) é a linguagem fundamental para a criação de páginas web, permitindo estruturar e apresentar conteúdo de forma acessível e organizada. Em meus projetos, utilizo HTML para desenvolver interfaces de usuário que são intuitivas e claras, garantindo que a experiência do usuário seja otimizada.

Css

CSS (Cascading Style Sheets) é uma linguagem de estilo fundamental que complementa o HTML, permitindo a estilização e a apresentação visual das páginas web. Em meus projetos, utilizo CSS para criar interfaces atraentes e responsivas, garantindo que o conteúdo não apenas seja acessível, mas também esteticamente agradável.

JavaScript

JavaScript é uma linguagem de programação essencial para o desenvolvimento web, permitindo a criação de páginas interativas e dinâmicas. Em meus projetos, utilizo JavaScript para adicionar funcionalidades que melhoram a experiência do usuário, como validação de formulários, manipulação de eventos e atualizações assíncronas de conteúdo.
